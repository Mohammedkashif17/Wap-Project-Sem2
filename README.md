# 🎬 Movie Watchlist App

## 📌 Project Description

The **Movie Watchlist App** is a web application that allows users to search for movies and manage a personalized watchlist. Users can explore movie details, add their favorite movies, and organize them easily.

This project demonstrates the use of JavaScript, API integration, and modern UI design.

---

## 🌐 API Used

This project uses the **OMDb API** to fetch movie data in real-time.

* API Link: https://www.omdbapi.com/

---

## 🚀 Features (Planned)

### 🔍 Search Functionality

* Users can search for movies by title
* Real-time results displayed dynamically

### ❤️ Watchlist Feature

* Add/remove movies from watchlist
* Data stored using localStorage

### 🎭 Filtering

* Filter movies by type (Movie / Series / Episode)

### 🔽 Sorting

* Sort movies by:

  * Year (Ascending / Descending)
  * Title (A-Z)

### 🎨 UI/UX

* Responsive design (Mobile, Tablet, Desktop)
* Clean and modern movie-card layout
